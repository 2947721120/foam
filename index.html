FOAM is a full-stack, reactive, deeply MVC, metaprogramming Javascript framework.


A more complete description can be found on the About page.


FOAM is beta: it is partially documented and evolving rapidly. But it's open source and is ready to use. The documentation is incomplete, but coverage is expanding all the time. API docs can be found here, and overviews in the wiki. The best place to start is the Tutorial. You can check out the source code from the github repository.
