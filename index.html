FOAM is a full-stack, reactive, deeply MVC, metaprogramming Javascript framework.


A more complete description can be found on the About page.


FOAM is alpha: it is sparsely documented and evolving rapidly. But it's open source and can be used by the brave. What documentation there is can be found on the main Github repo and wiki. The best place to start is the Tutorial.
