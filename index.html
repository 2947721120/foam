FOAM is a full-stack, deeply MVC metaprogramming framework. It happens to be written in Javascript, but can be used to generate code in any language.


A more complete description can be found on the About page.


FOAM is alpha: it is sparsely documented and evolving rapidly. But it's open source and can be used by the brave. What documentation there is can be found on the main Github repo and wiki.

  • Calculator
